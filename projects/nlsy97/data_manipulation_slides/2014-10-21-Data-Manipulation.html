Study in Focus

NLSY97

  • National Longitudinal Study of Youth 1997
  • Household sample
  • ~ 9,000 respondents
  • 5 cohorts
  • ages from 13 to 31
  • started in 1997
  • continues today
  • outcome:
    Church attendance

Working with NLS Investigator

To explore the variables in the native context of NLS,

  1. go to NLS Investigator (you will have to register a free account to keep track of your progress),
  2. select "NLYS97 1997-2011" in the first dropdown box and then
  3. click "Choose File" under "Upload Tagset."
  4. Select the file NLSY97_Attend_20141021.NLSY97 from the folder /Data/Extract/NLSY97_Attend_20141021. All the variables from this extract will be loaded into NLS Web Investigator.

Alternatively, one can locate the particular item of interest by copying and pasting its "Variable Title" it into "Word in Title" search line of the NLS Investigator, as indicated in the graphic below.

Using NLS Investigator a list of variables was downloaded from NLS datasets. All the downloaded materials were unzipped into the folder

/Data/Extracts/NLSY97_Attend_20141021

The downloaded zip.forlder included:

  • .cdb – codebook containing item descriptions
  • .csv – data in comma delimited format
  • .NLSY97 – tagset, the list of variables in the downloaded dataset
  • .dtc – STATA dictionary file of selected variables, contains data as well
